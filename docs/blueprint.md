# **App Name**: SaglikNet

## Core Features:

- Prescription Creation: Allow doctors to create and digitally sign prescriptions.
- Pharmacy Search: Enable users to search for nearby pharmacies based on location or medication availability.
- Real-time Prescription Verification: Verify prescriptions in real-time at pharmacies, checking for validity, interactions, and dosage.
- Inventory Management: Allow pharmacies to manage and update their medication inventory.
- Patient Profile Management: Allow patients to securely access their prescription history and manage their profiles.
- Smart Suggestion Tool: Leverage an AI tool to provide smart suggestions for medication dosages, refills, and potential interactions based on patient history.
- Secure Data Storage: Securely store prescription and patient data to comply with healthcare regulations.
- OTP System: Implement OTP verification via mail and phone number for enhanced security.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to inspire trust and security.
- Background color: Light gray (#F0F4F8) for a clean, professional look.
- Accent color: Teal (#26A69A) to highlight important actions and information.
- Body and headline font: 'Poppins', a sans-serif, for clear readability and a modern feel.
- Use consistent, recognizable icons for medication types, dosage instructions, and user actions.
- Design a clean, intuitive layout with clear hierarchies to ensure ease of use for doctors, pharmacists, and patients.
- Use subtle animations to guide users through the prescription process and provide feedback on actions.
